### 一、解构赋值的核心概念
解构赋值（Destructuring Assignment）是 ES6 新增的语法，它允许你**从数组或对象中提取值，并快速赋值给变量**，替代了传统的逐个赋值方式，让代码更简洁、可读性更高。

简单比喻：就像拆快递，你不需要把整个快递箱抱走，而是直接取出里面需要的物品，解构赋值就是“拆”数组/对象这个“快递箱”，只取需要的“物品”（值）。

### 二、数组解构赋值
数组解构是按**位置顺序**匹配值和变量，适用于有序的数据集合。

#### 1. 基础用法
```javascript
// 传统赋值方式
const arr = [1, 2, 3];
const a = arr[0];
const b = arr[1];
const c = arr[2];
console.log(a, b, c); // 1 2 3

// 解构赋值方式
const [x, y, z] = arr;
console.log(x, y, z); // 1 2 3
```

#### 2. 常见场景
+ **跳过不需要的值**：用逗号占位跳过特定位置的值

```javascript
const [first, , third] = [10, 20, 30];
console.log(first); // 10
console.log(third); // 30
```

+ **剩余值收集**：用 `...` 收集剩余的所有值（剩余运算符）

```javascript
const [a, ...rest] = [1, 2, 3, 4];
console.log(a); // 1
console.log(rest); // [2, 3, 4]
```

+ **默认值**：当数组对应位置无值（或为 `undefined`）时，使用默认值

```javascript
const [a = 0, b = 0] = [5];
console.log(a); // 5（有值则用原值）
console.log(b); // 0（无值则用默认值）
```

+ **交换变量**：无需临时变量，一行交换两个变量

```javascript
let x = 1, y = 2;
[x, y] = [y, x];
console.log(x, y); // 2 1
```

### 三、对象解构赋值
对象解构是按**属性名**匹配值和变量（与顺序无关），适用于键值对形式的数据。

#### 1. 基础用法
```javascript
// 传统赋值方式
const obj = { name: "张三", age: 20 };
const name = obj.name;
const age = obj.age;
console.log(name, age); // 张三 20

// 解构赋值方式
const { name: userName, age: userAge } = obj; // 重命名变量（name→userName，age→userAge）
console.log(userName, userAge); // 张三 20

// 简化写法（变量名与属性名一致）
const { name, age } = obj;
console.log(name, age); // 张三 20
```

#### 2. 常见场景
+ **默认值**：当对象无对应属性（或属性值为 `undefined`）时，使用默认值

```javascript
const { name, gender = "男" } = { name: "李四" };
console.log(gender); // 男
```

+ **嵌套对象解构**：解构多层嵌套的对象

```javascript
const user = {
  info: {
    basic: {
      id: 1,
      name: "王五"
    }
  }
};
// 嵌套解构：逐层匹配属性名
const { info: { basic: { id, name } } } = user;
console.log(id, name); // 1 王五
```

+ **剩余值收集**：用 `...` 收集对象剩余的属性

```javascript
const { a, ...rest } = { a: 1, b: 2, c: 3 };
console.log(a); // 1
console.log(rest); // { b: 2, c: 3 }
```

+ **函数参数解构**：直接解构函数的对象参数，简化参数使用

```javascript
// 传统写法
function printUser(user) {
  console.log(user.name, user.age);
}
printUser({ name: "赵六", age: 25 }); // 赵六 25

// 解构写法
function printUser({ name, age = 18 }) { // 支持默认值
  console.log(name, age);
}
printUser({ name: "赵六" }); // 赵六 18
```

#### 3. 字符串解构
```javascript
let [a, b, c] = "hello";
      console.log(a); //h
      console.log(b); //e
      console.log(c); //l
 let { length } = "hello";
      console.log(length);//5
```

### 四、注意事项
1. **解构空值**：如果解构的目标是 `null` 或 `undefined`，会报错，需提前判断

```javascript
const { name } = null; // Uncaught TypeError: Cannot destructure property 'name' of 'null' as it is null.
// 解决方案：先判断是否为有效值
const obj = null;
const { name } = obj || {}; // 无报错，name 为 undefined
```

2. **变量声明与解构分开**：如果变量已声明，解构时不能直接写 `{ a, b } = obj`，需加括号

```javascript
let a, b;
// { a, b } = { a: 1, b: 2 }; // 报错
({ a, b } = { a: 1, b: 2 }); // 正确
```

3. **数组解构的顺序性**：数组解构依赖位置，对象解构依赖属性名，不要混淆。

### 总结
1. 解构赋值的核心是**提取值并快速赋值**，分为数组解构（按位置）和对象解构（按属性名）两大类。
2. 常用技巧：剩余值收集（`...`）、默认值、嵌套解构、函数参数解构，能大幅简化代码。
3. 注意避坑：解构 `null/undefined` 会报错，已声明变量解构需加括号。
