# 第三方登录知识库

本目录沉淀 `wisdomCampusApp`（小程序）与 `wisdomCampusClient（PC）` 两个项目的第三方登录文档，按平台、容器和接入阶段拆分，便于后续复用。

## 建议阅读顺序

1. 先看 [00-概念速查/第三方登录-术语与场景速查.md](./00-概念速查/第三方登录-术语与场景速查.md)，先分清 H5 微应用、PC 扫码登录、小程序三个场景。
2. 再按平台进入 [01-钉钉/](./01-钉钉/) 或 [02-企业微信/](./02-企业微信/)。
3. 需要做方案选型或核对差异时，再看 [99-对比笔记/钉钉与企业微信登录实现笔记.md](./99-对比笔记/钉钉与企业微信登录实现笔记.md)。
