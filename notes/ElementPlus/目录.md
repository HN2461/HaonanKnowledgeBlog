# Element Plus 专题目录

## 为什么单独开这个目录

`Element Plus` 不是普通的 CSS 补充知识。

它更像是：

- `Vue 3` 生态里常见的 `PC` 端组件库
- 后台管理系统里高频出现的一整套现成组件能力
- 和移动端组件库分工完全不同的一类前端工具

所以这里单独开一个目录，专门整理 `Element Plus`。

---

## 当前内容

- 第一篇《快速认识 Element Plus：为什么它是 Vue 3 里常见的 PC 端组件库》
  - 先讲清它的定位、最新官方版本、Vue 3 + Vite 接入方式、按需导入、主题定制、国际化和暗黑模式
- 第二篇《安装与引入：把 Element Plus 真正接进 Vue 3 项目》
  - 详细拆开安装、完整引入、自动按需导入、手动导入、Volar 支持和全局配置
- 第三篇《基础组件：Button、Icon、Container、Space、Text 怎么搭出像样的后台界面》
  - 专门讲按钮分层、图标搭配、页面骨架、间距统一和文字收口
- 第四篇《表单体系：Form、Input、Select、DateTimePicker、Upload 怎么配合》
  - 讲中后台表单录入、筛选、校验、上传和时间范围选择
- 第五篇《中后台核心组件：Table、Pagination、Dialog、Drawer、Message 怎么配合》
  - 讲列表页的主循环：展示、翻页、编辑和反馈
- 第六篇《主题定制、国际化、暗黑模式、ConfigProvider：Element Plus 全局能力怎么管》
  - 讲语言、Day.js 本地化、主题变量、SCSS 主题和暗黑模式的统一管理

---

## 建议阅读顺序

1. 第一篇：先知道 Element Plus 是什么、适合什么项目
2. 第二篇：把它正确接进 Vue 3 + Vite 项目
3. 第三篇：先把最基础的按钮、图标、布局和间距用顺
4. 第四篇：再进表单体系
5. 第五篇：再进表格、分页、弹层和消息反馈
6. 第六篇：最后统一处理主题、语言和暗黑模式
