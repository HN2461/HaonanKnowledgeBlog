## 前端转后端学习里程碑

1. **先把运行环境跑通**：第一篇 + 第五篇，装好 Node.js 和 nvm，理解模块系统和 npm 基本命令
2. **补齐服务端文件与路径意识**：第二篇把 `fs`、`path`、Stream 吃透
3. **建立请求-响应模型**：第三篇用原生 `http` 自己写服务端
4. **开始写可维护的接口项目**：第四篇 + 第六篇，先会拆模块，再上 Express
5. **把数据持久化接上**：第七篇详解系列，MongoDB + Mongoose 全套
6. **把登录态和安全补上**：第八篇详解系列，Cookie/Session/JWT + 安全实践

## 建议的练手项目顺序

1. `hello-node`：输出环境信息，练习命令行运行和调试
2. `file-tool`：本地文件扫描/整理脚本，练习 `fs`、`path`、Stream
3. `mini-http-server`：不用框架，手写 GET/POST 接口
4. `express-notes-api`：Express + 路由拆分 + 统一错误处理
5. `auth-api`：MongoDB + Mongoose + JWT + bcrypt，完成注册、登录、鉴权
