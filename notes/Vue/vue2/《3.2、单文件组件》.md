# 3.2、单文件组件
## 3.2.1. 什么是单文件组件？
1. **一个文件对应一个组件**  
（之前我们所学的是非单文件组件，一个HTML文件中定义了多个组件）
2. **文件扩展名**  
单文件组件的名字通常是：`x.vue`，这是Vue框架规定的。只有Vue框架能够识别，浏览器无法直接打开运行。需要Vue框架进行编译，将`x.vue`最终编译为浏览器能识别的HTML、JS、CSS。
3. **文件名命名规范**  
单文件组件的文件名命名规范和组件名的命名规范相同：
    - 全部小写：`userlist`  
    - 首字母大写，后面全部小写：`Userlist`  
    - kebab-case命名法：`user-list`  
    - CamelCase命名法：`UserList`（推荐使用这种方式，与Vue开发者工具呼应）  
注意：不能用小驼峰：`userList`；不能用HTML标签名。

## 3.2.2. `x.vue`文件的内容包括三块：
1. **结构**  
`<template>HTML代码</template>`  
2. **交互**  
`<script>JS代码</script>`  
3. **样式**  
`<style>CSS代码</style>`

## 3.2.3. `export`和`import`，ES6的模块化语法
1. **使用**`export`**导出（暴露）组件**  
在需要使用组件的`x.vue`文件中使用`import`导入组件：
    - **默认导入和导出**  

```javascript
export default {}
import 任意名称 from '模块标识符'
```

    - **按需导入和导出**  

```javascript
export {a, b}
import {a, b} from '模块标识符'
```

    - **分别导出**  

```javascript
export var name = 'zhangsan'
export function sayHi(){}
```

**导入**  

```javascript
import {name, sayHi} from '模块标识符'
```
