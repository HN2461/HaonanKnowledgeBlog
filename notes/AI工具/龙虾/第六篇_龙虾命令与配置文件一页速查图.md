# 第六篇：龙虾命令与配置文件一页速查图

> 适用版本：`openclaw-cn.cmd 0.1.7`  
> 使用方式：每天开机先看这页，遇错先对照这页。  
> 详细解释：命令细节看第五篇，重启排障看第二篇。

---

## 1. 每天最小闭环（4 条命令）

```powershell
openclaw-cn.cmd gateway run
openclaw-cn.cmd gateway status --json
openclaw-cn.cmd health --json
openclaw-cn.cmd status --all
```

判断标准：

1. `gateway status --json` 里 `rpc.ok=true`
2. `health --json` 里 `ok=true`

---

## 2. 最常用命令速查

| 任务 | 命令 | 成功判据 |
|---|---|---|
| 看版本 | `openclaw-cn.cmd --version` | 返回版本号 |
| 前台启动 | `openclaw-cn.cmd gateway run` | 终端持续运行 |
| 后台安装 | `openclaw-cn.cmd gateway install` | 返回 installed/registered |
| 后台启动 | `openclaw-cn.cmd gateway start` | `rpc.ok=true` |
| 停止后台 | `openclaw-cn.cmd gateway stop` | 返回 stopped |
| 卸载后台 | `openclaw-cn.cmd gateway uninstall` | 返回 uninstalled |
| 全量诊断 | `openclaw-cn.cmd status --all` | 输出诊断表 |
| 实时日志 | `openclaw-cn.cmd logs --follow` | 持续输出日志 |
| 自动修复 | `openclaw-cn.cmd doctor --fix` | 输出修复结果 |
| 安全审计 | `openclaw-cn.cmd security audit --json` | 输出告警清单 |

## 3. 配置文件速查（必须认识）

| 文件/目录 | 默认路径 | 作用 |
|---|---|---|
| 主配置 | `C:\Users\<你>\.openclaw\openclaw.json` | 网关端口、bind、workspace、认证 |
| 服务脚本 | `C:\Users\<你>\.openclaw\gateway.cmd` | 计划任务自动模式调用入口 |
| 工作区 | `C:\Users\<你>\.openclaw\workspace\` | 业务资料与规则文件 |
| SOUL | `<workspace>\SOUL.md` | 助手语气、边界、主动性 |
| USER | `<workspace>\USER.md` | 你的目标、习惯、偏好 |
| AGENTS | `<workspace>\AGENTS.md` | 执行规则与风险处理 |
| HEARTBEAT | `<workspace>\HEARTBEAT.md` | 心跳巡检任务清单 |
| 会话存储 | `...\agents\main\sessions\sessions.json` | 会话上下文元数据 |

## 4. 先查再改（推荐顺序）

```powershell
openclaw-cn.cmd gateway status --json
openclaw-cn.cmd status --all
openclaw-cn.cmd health --json
openclaw-cn.cmd logs --follow
```

只有在看完状态后再执行修复命令，不要直接“盲重装”。

## 5. 报错 -> 立即动作

### 5.1 `gateway closed (1006 abnormal closure ...)`

```powershell
openclaw-cn.cmd gateway status --json
openclaw-cn.cmd gateway run
```

### 5.2 `connect ECONNREFUSED 127.0.0.1:18789`

```powershell
netstat -ano | findstr :18789
openclaw-cn.cmd gateway run
```

### 5.3 `service loaded but rpc false`

```powershell
openclaw-cn.cmd gateway stop
openclaw-cn.cmd gateway uninstall
openclaw-cn.cmd gateway install --force
openclaw-cn.cmd gateway start
openclaw-cn.cmd gateway status --json
```

### 5.4 `LastTaskResult = 1`（Openclaw Gateway）

```powershell
Get-ScheduledTaskInfo -TaskName 'Openclaw Gateway' | Format-List LastRunTime,LastTaskResult
Test-Path C:\Users\HN246\.openclaw\gateway.cmd
```

如果 `gateway.cmd` 不存在：优先按第二篇“重建服务入口”执行。
