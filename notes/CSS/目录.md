### C.1 布局片段

#### 水平垂直居中
```css
/* 方案1：Flexbox */
.center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}

/* 方案2：Grid */
.center-grid {
  display: grid;
  place-items: center;
}

/* 方案3：绝对定位 + Transform */
.center-absolute {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

/* 方案4：绝对定位 + margin */
.center-margin {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  margin: auto;
  width: 200px;  /* 需要指定宽高 */
  height: 100px;
}
```

#### 两列布局
```css
/* 左固定右自适应 */
.two-column-left-fixed {
  display: flex;
}

.sidebar {
  width: 200px;
  flex-shrink: 0;
}

.main {
  flex: 1;
}

/* 左右固定中间自适应 */
.three-column {
  display: flex;
}

.sidebar-left,
.sidebar-right {
  width: 200px;
  flex-shrink: 0;
}

.content {
  flex: 1;
}
```

#### 响应式网格
```css
/* 自适应网格 */
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 20px;
}

/* 断点网格 */
.breakpoint-grid {
  display: grid;
  gap: 20px;
  grid-template-columns: 1fr;
}

@media (min-width: 768px) {
  .breakpoint-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (min-width: 1024px) {
  .breakpoint-grid {
    grid-template-columns: repeat(3, 1fr);
  }
```
