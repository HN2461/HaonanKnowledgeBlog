# CatPaw（猫爪）公开资料整理

这组笔记只保留公开资料里能核实的内容，重点参考：

- CatPaw 官网首页
- CatPaw 公开用户手册
- 美团技术团队 2025 年公开文章中的 CatPaw 介绍

也就是说：

- 保留能在官网或公开手册里找到的能力说明
- 删除或不再展开没有公开证据支撑的“内网平台深度集成”“一长串 CLI 命令”“插件包名”等推测性内容
- 遇到官网和手册写法不完全一致的地方，优先写“能确定的事实”，不硬猜

## 一句话先认识 CatPaw

公开资料里，CatPaw 的官方定位是：

- 美团推出的 `AI 编程 Agent`
- 以 `IDE` 形态集成
- 核心能力围绕 `Tab 补全`、`Ask / Agent 对话`、`项目预览调试`、`代码库索引`、`Rules`、`MCP`

补充两条能公开核实的时间线：

- CatPaw 官网结构化数据里标注的 `datePublished` 是 `2025-11-04`
- 美团技术团队在 `2025-12-29` 的年度汇总文章中提到：CatPaw 早在 `2023 年` 已以编辑器插件形态在美团内部上线，2025 年完成升级后开启公测；内部研发渗透率超过 `95%`，增量代码 AI 生成率超过 `50%`
