## 覆盖主题

- 工具定位：跨平台桌面 All-in-One AI CLI 配置管理中控台
- 支持工具：Claude Code、Codex CLI、Gemini CLI、OpenCode、OpenClaw、Hermes Agent（v3.14.0+）
- Provider 管理：新增、切换、速度测试（Stream Check）、故障转移、一键回滚
- 本地路由（Local Routing）：热切换、自动故障转移、熔断器、请求日志（v3.9.0+）
- MCP 服务器：跨应用统一管理、双向同步
- 系统提示词：多预设、跨应用（CLAUDE.md / AGENTS.md / GEMINI.md）
- Skills 管理：GitHub 仓库自动扫描与一键安装、ZIP 安装、批量更新
- 会话管理器：浏览、搜索、恢复全部应用对话历史
- 用量追踪：跨 Provider Token 消耗与费用统计
- 云同步：Dropbox / OneDrive / iCloud / WebDAV
- Deep Link：ccswitch:// 协议一键导入配置
