## 这一轮重点更新了什么

- 模式与权限：补入 `auto mode`、`bypassPermissions`、`dontAsk` 的当前口径，修正 `acceptEdits` 现在会自动放行常见文件系统命令的事实。
- 会话与上下文：补入 `/context`、`/recap`、`/schedule`、`/loop`、`/autofix-pr` 等 2026-04 新能力。
- 记忆与规则：补入 `CLAUDE.local.md`、`MEMORY.md` 的 `200 行或 25KB` 加载边界，以及 `AGENTS.md` 通过 `@AGENTS.md` 复用的方式。
- 技能与插件：修正 Skill 目录结构为 `.../skills/<name>/SKILL.md`，并补入 `/reload-plugins` 和插件命名空间规则。
- 模型与配置：修正设置优先级、`/effort`、`xhigh`、1M 上下文与 200K 之后“无额外 premium” 的当前说明。
- 第三方接入：把智谱官方文档里的最新模型映射更新为 `GLM-5.1` / `glm-5-turbo` 口径。

## 覆盖主题

- 快速开始：安装、登录、第一轮提问、第一轮改动、代理循环
- 模式与权限：`default`、`acceptEdits`、`plan`、`auto`、`bypassPermissions`、`dontAsk`
- 会话与上下文：`/resume`、`/compact`、`/context`、`/tasks`、`/recap`
- 扩展体系：`CLAUDE.md`、`CLAUDE.local.md`、rules、skills、subagents、agent teams、hooks、MCP、plugins
- 工作流：理解代码库、修 bug、重构、测试、PR、批量改造、后台任务
- 新功能：`/effort`、`/loop`、`/schedule`、`/autofix-pr`、`/reload-plugins`、Monitor tool
- 设置与个性化：settings、permissions、sandboxing、model config、fast mode、statusline、keybindings
- 第三方模型接入：GLM Coding Plan、模型映射、IDE 配置、版本兼容
