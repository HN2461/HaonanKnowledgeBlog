# 系统与文件目录

这个专题专门放“Windows 里的系统目录、用户目录、文件显示设置、路径变量、C盘清理”这类内容。

如果你经常折腾电脑配置，建议按下面顺序读，会比较顺：

## 推荐阅读顺序

1. [Windows C盘常见系统文件夹说明](./Windows%20C盘常见系统文件夹说明.md)
2. [Windows 用户目录、Public、AppData、ProgramData 详解](./Windows%20用户目录、Public、AppData、ProgramData%20详解.md)
3. [Windows 文件扩展名、隐藏文件与资源管理器显示设置](./Windows%20文件扩展名、隐藏文件与资源管理器显示设置.md)
4. [Windows C盘空间清理：Storage Sense、临时文件与 Windows.old](./Windows%20C盘空间清理：Storage%20Sense、临时文件与%20Windows.old.md)
5. [Windows 常见系统路径与环境变量速查](./Windows%20常见系统路径与环境变量速查.md)
6. [Windows 管理员权限、UAC 与“以管理员身份运行”](./Windows%20管理员权限、UAC%20与以管理员身份运行.md)
7. [Windows 启动项、任务管理器、MSConfig 与常用系统配置工具](./Windows%20启动项、任务管理器、MSConfig%20与常用系统配置工具.md)

## 这个专题覆盖什么

- `C:\\Windows`、`System32`、`SysWOW64`
- `Program Files`、`Program Files (x86)`
- `Users`、`Public`、`AppData`、`ProgramData`
- 文件扩展名、隐藏文件、资源管理器常用显示选项
- `Storage Sense`、临时文件、`Windows.old`
- `%USERPROFILE%`、`%APPDATA%`、`%LOCALAPPDATA%` 这类常见路径变量
- 管理员权限、`UAC`、“以管理员身份运行”
- 启动项、任务管理器、`msconfig`、常见系统配置工具

## 快速查找

- 想先搞懂 C 盘目录都是什么：看 [Windows C盘常见系统文件夹说明](./Windows%20C盘常见系统文件夹说明.md)
- 想弄懂 `Users`、`Public`、`AppData`、`ProgramData`：看 [Windows 用户目录、Public、AppData、ProgramData 详解](./Windows%20用户目录、Public、AppData、ProgramData%20详解.md)
- 想把隐藏文件、扩展名显示出来：看 [Windows 文件扩展名、隐藏文件与资源管理器显示设置](./Windows%20文件扩展名、隐藏文件与资源管理器显示设置.md)
- 想清理 C 盘但怕删错：看 [Windows C盘空间清理：Storage Sense、临时文件与 Windows.old](./Windows%20C盘空间清理：Storage%20Sense、临时文件与%20Windows.old.md)
- 想写路径、配脚本、找配置目录：看 [Windows 常见系统路径与环境变量速查](./Windows%20常见系统路径与环境变量速查.md)
- 想搞懂为什么明明是管理员还会弹权限确认：看 [Windows 管理员权限、UAC 与“以管理员身份运行”](./Windows%20管理员权限、UAC%20与以管理员身份运行.md)
- 想管理开机自启、排查启动冲突：看 [Windows 启动项、任务管理器、MSConfig 与常用系统配置工具](./Windows%20启动项、任务管理器、MSConfig%20与常用系统配置工具.md)
