### `includes()` - 检查是否包含子串
1. **<font style="color:rgb(0, 0, 0);">返回值</font>**<font style="color:rgb(0, 0, 0);">：找到子串返回</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">true</font>`<font style="color:rgb(0, 0, 0);">，未找到返回</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">false</font>`<font style="color:rgb(0, 0, 0);">；</font>
2. **<font style="color:rgb(0, 0, 0);">区分大小写</font>**<font style="color:rgb(0, 0, 0);">：对字母大小写敏感（如</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">'Hello'</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">不包含</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">'hello'</font>`<font style="color:rgb(0, 0, 0);">）；</font>
3. **<font style="color:rgb(0, 0, 0);">可选参数</font>**<font style="color:rgb(0, 0, 0);">：第二个参数可指定查找的起始位置（默认从索引 0 开始）；</font>
4. **<font style="color:rgb(0, 0, 0);">支持空字符串</font>**<font style="color:rgb(0, 0, 0);">：查找空字符串 </font>`<font style="color:rgb(0, 0, 0);">''</font>`<font style="color:rgb(0, 0, 0);"> 时始终返回 </font>`<font style="color:rgb(0, 0, 0);">true</font>`<font style="color:rgb(0, 0, 0);">（ECMAScript 规范）。</font>

```javascript
console.log(str.includes('world')); // false (区分大小写)

// 实际应用

    return email.includes('@') && email.includes('.');
}
```

## 字符串截取
### `slice()` - 提取子字符串
+ `<font style="color:rgb(0, 0, 0);">startIndex</font>`<font style="color:rgb(0, 0, 0);">：必需，提取的起始位置（索引从 0 开始）；若为负数，从字符串末尾倒着数（如</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">-1</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">表示最后一个字符）。</font>
+ `<font style="color:rgb(0, 0, 0);">endIndex</font>`<font style="color:rgb(0, 0, 0);">：可选，提取的结束位置（不包含该位置的字符）；省略则提取到字符串末尾；若为负数，同样从末尾倒着数。</font>
+ `<font style="color:rgb(0, 0, 0);">slice()</font>`<font style="color:rgb(0, 0, 0);"> 若 </font>`<font style="color:rgb(0, 0, 0);">start > end</font>`<font style="color:rgb(0, 0, 0);"> 返回空字符串，</font>`<font style="color:rgb(0, 0, 0);">substring()</font>`<font style="color:rgb(0, 0, 0);"> 会自动交换两者位置（如 </font>`<font style="color:rgb(0, 0, 0);">substring(4,1)</font>`<font style="color:rgb(0, 0, 0);"> 等价于 </font>`<font style="color:rgb(0, 0, 0);">substring(1,4)</font>`<font style="color:rgb(0, 0, 0);">）。</font>
+ <font style="color:rgba(0, 0, 0, 0.8);">当交换 </font>`<font style="color:rgba(0, 0, 0, 0.8);">slice()</font>`<font style="color:rgba(0, 0, 0, 0.8);"> 的参数（即 </font>`<font style="color:rgba(0, 0, 0, 0.8);">start > end</font>`<font style="color:rgba(0, 0, 0, 0.8);">）时，</font>**<font style="color:rgba(0, 0, 0, 0.95) !important;">无论参数是正数、负数，最终都会返回空数组 </font>**`**<font style="color:rgb(0, 0, 0);">[]</font>**`<font style="color:rgba(0, 0, 0, 0.8);">，且原数组完全不受影响。</font>

```javascript
const str = 'Hello World';
console.log(str.slice(0, 5)); // 'Hello' (从0到5，不包括5)
console.log(str.slice(6)); // 'World' (从6到末尾)
console.log(str.slice(-5)); // 'World' (负值表示从末尾开始)
console.log(str.slice(3, 7)); // 'lo W'
console.log(str.slice(7, 3)); // '' (如果开始>结束，返回空字符串)
```

### `substring()` - 提取子字符串
+ <font style="color:rgb(0, 0, 0);">自动修正参数顺序：若</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">indexStart > indexEnd</font>`<font style="color:rgb(0, 0, 0);">，方法会自动交换两者（如</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">substring(5, 2)</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">等价于</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">substring(2, 5)</font>`<font style="color:rgb(0, 0, 0);">）。</font>
+ <font style="color:rgb(0, 0, 0);">处理负数参数：若参数为负数或</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">NaN</font>`<font style="color:rgb(0, 0, 0);">，会被当作</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">0</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">处理（如</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">substring(-2, 3)</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">等价于</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">substring(0, 3)</font>`<font style="color:rgb(0, 0, 0);">）。</font>
+ <font style="color:rgb(0, 0, 0);">超出范围处理：若参数大于字符串长度，按字符串长度计算（如 </font>`<font style="color:rgb(0, 0, 0);">substring(10, 20)</font>`<font style="color:rgb(0, 0, 0);"> 若原串长度为 15，等价于 </font>`<font style="color:rgb(0, 0, 0);">substring(10, 15)</font>`<font style="color:rgb(0, 0, 0);">）。</font>

```javascript
const str = 'Hello World';
console.log(str.substring(0, 5)); // 'Hello'
console.log(str.substring(6)); // 'World'
console.log(str.substring(3, 7)); // 'lo W'

// 与slice的区别
console.log(str.substring(-3, 5)); // 'Hello' (负值视为0)
console.log(str.substring(7, 3)); // 'lo W' (会自动交换参数)
```

## 字符串修改
### `replace()` - 替换字符串
```javascript
str.replace(匹配目标, 替换内容)
```

+ <font style="color:rgb(0, 0, 0);">匹配目标：可是「字符串」（仅匹配首次出现）或「正则表达式」（支持全局 / 忽略大小写等规则）</font>
+ <font style="color:rgb(0, 0, 0);">替换内容：可是「字符串」或「回调函数」（处理复杂替换逻辑）</font>

```javascript
const str = 'Hello World, Hello Universe';
console.log(str.replace('Hello', 'Hi')); // 'Hi World, Hello Universe'
console.log(str.replace(/Hello/g, 'Hi')); // 'Hi World, Hi Universe'

// 使用函数替换
const result = 'Hello 123 World 456'.replace(/\d+/g, (match) => {
    return parseInt(match) * 2;
});
console.log(result); // 'Hello 246 World 912'

// 使用捕获组
const date = '2024-01-15';
const newDate = date.replace(/(\d{4})-(\d{2})-(\d{2})/, '$2/$3/$1');
console.log(newDate); // '01/15/2024'
```

### `replaceAll()` - 替换所有匹配项
+ <font style="color:rgb(0, 0, 0);">对比 </font>`<font style="color:rgb(0, 0, 0);">replace()</font>`<font style="color:rgb(0, 0, 0);">（仅替换首次匹配项），</font>`<font style="color:rgb(0, 0, 0);">replaceAll()</font>`<font style="color:rgb(0, 0, 0);"> 会替换字符串中</font>**<font style="color:rgb(0, 0, 0);">所有符合条件的子串</font>**<font style="color:rgb(0, 0, 0);">，无需依赖正则全局标志（</font>`<font style="color:rgb(0, 0, 0);">g</font>`<font style="color:rgb(0, 0, 0);">）。</font>

```javascript

console.log(str.replaceAll('Hello', 'Hi')); // 'Hi World, Hi Universe'

// 实际应用：批量替换
const text = 'apple, banana, apple, orange, apple';
const newText = text.replaceAll('apple', 'pear');
console.log(newText); // 'pear, banana, pear, orange, pear'
```

### `repeat()` - 重复字符串
<font style="color:rgb(0, 0, 0);">按</font>**<font style="color:#DF2A3F;">指定次数复制原字符串</font>**<font style="color:rgb(0, 0, 0);">并拼接返回，</font>**<font style="color:rgb(0, 0, 0);">不改变原字符串</font>**<font style="color:rgb(0, 0, 0);">（因字符串不可变），属于字符串修改类方法。</font>

1. **<font style="color:rgb(0, 0, 0);">语法</font>**<font style="color:rgb(0, 0, 0);">：</font>`<font style="color:rgb(0, 0, 0);">str.repeat(count)</font>`<font style="color:rgb(0, 0, 0);">，参数</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">count</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">为非负整数（表示重复次数）。</font>
2. **<font style="color:rgb(0, 0, 0);">返回值</font>**<font style="color:rgb(0, 0, 0);">：新字符串，由原字符串重复</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">count</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">次组成。</font>
3. **<font style="color:rgb(0, 0, 0);">边界处理</font>**<font style="color:rgb(0, 0, 0);">：</font>
    - `<font style="color:rgb(0, 0, 0);">count = 0</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">时，返回空字符串；</font>
    - `<font style="color:rgb(0, 0, 0);">count</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">为小数时，会向下取整（如</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">2.9</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">视为</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">2</font>`<font style="color:rgb(0, 0, 0);">）；</font>
    - <font style="color:rgb(0, 0, 0);">不支持负数、 Infinity，否则会报错。</font>

```javascript
console.log('Hello '.repeat(3)); // 'Hello Hello Hello '
console.log('Hi'.repeat(2.5)); // 'HiHi' (2.5会被转换为2)
console.log('Hi'.repeat(0)); // ''
console.log('Hi'.repeat(3.9)); // 'HiHiHi' (向下取整)

// 实际应用：生成分隔线
function createSeparator(length, char = '-') {
    return char.repeat(length);
}
console.log(createSeparator(20)); // '--------------------'
```

### `split()` - 分割字符串为数组
1. **<font style="color:rgb(0, 0, 0);">基础用法</font>**<font style="color:rgb(0, 0, 0);">：接收一个分隔符（字符串或正则表达式），按该分隔符拆分字符串。</font>
    - <font style="color:rgb(0, 0, 0);">示例：</font>`<font style="color:rgb(0, 0, 0);">'a,b,c'.split(',')</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">→</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">['a', 'b', 'c']</font>`
2. **<font style="color:rgb(0, 0, 0);">可选参数（长度限制）</font>**<font style="color:rgb(0, 0, 0);">：第二个参数可指定返回数组的最大长度，超出部分会被忽略。</font>
    - <font style="color:rgb(0, 0, 0);">示例：</font>`<font style="color:rgb(0, 0, 0);">'a,b,c,d'.split(',', 2)</font>`<font style="color:rgb(0, 0, 0);"> → </font>`<font style="color:rgb(0, 0, 0);">['a', 'b']</font>`

```javascript
const str = 'apple,banana,orange';
console.log(str.split(',')); // ['apple', 'banana', 'orange']
console.log(str.split(',', 2)); // ['apple', 'banana'] (限制分割次数)

// 使用正则表达式分割
const str2 = 'apple, banana; orange';
console.log(str2.split(/[,;]\s*/)); // ['apple', 'banana', 'orange']

// 分割每个字符
console.log('Hello'.split('')); // ['H', 'e', 'l', 'l', 'o']

// 实际应用：解析CSV
const csv = 'John,Doe,30,New York';
const [firstName, lastName, age, city] = csv.split(',');
```

## 字符串填充
### `padStart()` - 在开头填充字符串
**<font style="color:rgb(0, 0, 0);">在字符串开头（左侧）补充指定字符</font>**<font style="color:rgb(0, 0, 0);">，直到字符串达到目标长度，返回新字符串（不修改原字符串，因字符串不可变）。</font>

```javascript
str.padStart(targetLength [, padString])
```

+ **<font style="color:rgb(0, 0, 0);">参数 1（必选）</font>**<font style="color:rgb(0, 0, 0);">：</font>`<font style="color:rgb(0, 0, 0);">targetLength</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">- 最终想要的字符串总长度。若原字符串长度 ≥ 该值，直接返回原字符串。</font>
+ **<font style="color:rgb(0, 0, 0);">参数 2（可选）</font>**<font style="color:rgb(0, 0, 0);">：</font>`<font style="color:rgb(0, 0, 0);">padString</font>`<font style="color:rgb(0, 0, 0);"> - 用于填充的字符（默认是空格）。若该字符串长度超过需要填充的长度，会被截断取前半部分。</font>

```javascript
console.log('5'.padStart(3, '0')); // '005'
console.log('123'.padStart(5, '0')); // '00123'
console.log('hello'.padStart(10, '*-')); // '*-*-*hello'

// 实际应用：格式化数字
function formatNumber(num, length = 8) {
    return num.toString().padStart(length, '0');
}
console.log(formatNumber(123)); // '00000123'
```

### `padEnd()` - 在结尾填充字符串
```javascript
console.log('5'.padEnd(3, '0')); // '500'
console.log('hello'.padEnd(10, '!')); // 'hello!!!!!'
console.log('hello'.padEnd(10, ' world')); // 'hello worl'

// 实际应用：对齐文本
const items = [
    { name: 'apple', price: 1.2 },
    { name: 'banana', price: 0.5 },
    { name: 'orange', price: 0.8 }
];

items.forEach(item => {
    const name = item.name.padEnd(10, '.');
    const price = item.price.toFixed(2).padStart(6, ' ');
    console.log(`${name}${price}`);
});
```

## 字符串比较与验证
### `localeCompare()` - 区域敏感字符串比较
```javascript
console.log('a'.localeCompare('b')); // -1 (a在b之前)
console.log('b'.localeCompare('a')); // 1 (b在a之后)
console.log('a'.localeCompare('a')); // 0 (相等)

// 考虑区域设置
console.log('ä'.localeCompare('z', 'de')); // -1 (在德语中ä排在a之后)
console.log('ä'.localeCompare('z', 'sv')); // 1 (在瑞典语中ä排在z之后)

// 实际应用：排序

words.sort((a, b) => a.localeCompare(b, 'en', { sensitivity: 'base' }));
console.log(words); // ['apple', 'Äpfel', 'Banana', 'cherry']
```

### `match()` - 正则表达式匹配
**<font style="color:rgb(0, 0, 0);">用正则表达式匹配字符串</font>**<font style="color:rgb(0, 0, 0);">，返回匹配结果（数组或 </font>`<font style="color:rgb(0, 0, 0);">null</font>`<font style="color:rgb(0, 0, 0);">），具体解析如下：</font>

<font style="color:rgb(0, 0, 0);">语法：</font>`<font style="color:rgb(0, 0, 0);">str.match(regexp)</font>`

+ <font style="color:rgb(0, 0, 0);">参数</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">regexp</font>`<font style="color:rgb(0, 0, 0);">：正则表达式（可直接写</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">/规则/</font>`<font style="color:rgb(0, 0, 0);">，或用</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">new RegExp()</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">创建）；若传入非正则值，会自动转为正则。</font>
+ <font style="color:rgb(0, 0, 0);">返回值：</font>
    - <font style="color:rgb(0, 0, 0);">匹配成功：返回数组（数组 [0] 是完整匹配字符串，后续元素是正则分组捕获的内容）；</font>
    - <font style="color:rgb(0, 0, 0);">匹配失败：返回 </font>`<font style="color:rgb(0, 0, 0);">null</font>`<font style="color:rgb(0, 0, 0);">。</font>

```javascript
const str = 'The rain in Spain falls mainly in the plain';
console.log(str.match(/ain/g)); // ['ain', 'ain', 'ain']
console.log(str.match(/ain/)); // ['ain', index: 5, input: 'The rain...', groups: undefined]

// 使用捕获组
const dateStr = '2024-01-15';
const match = dateStr.match(/(\d{4})-(\d{2})-(\d{2})/);
if (match) {
    console.log(`年: ${match[1]}, 月: ${match[2]}, 日: ${match[3]}`);
}
```

### `matchAll()` - 返回所有匹配的迭代器

```javascript
// [

```

### `toString()` 和 `valueOf()`
<font style="color:rgb(0, 0, 0);">1. toString()</font>

+ **<font style="color:rgb(0, 0, 0);">作用</font>**<font style="color:rgb(0, 0, 0);">：将 String 对象转换为原始字符串（与字符串字面量格式一致）。</font>
+ **<font style="color:rgb(0, 0, 0);">使用场景</font>**<font style="color:rgb(0, 0, 0);">：当需要将 String 实例（通过 </font>`<font style="color:rgb(0, 0, 0);">new String()</font>`<font style="color:rgb(0, 0, 0);"> 创建的对象）转为原始字符串时自动调用，也可手动调用。</font>

<font style="color:rgb(0, 0, 0);">2. valueOf()</font>

+ **<font style="color:rgb(0, 0, 0);">作用</font>**<font style="color:rgb(0, 0, 0);">：返回 String 对象的原始值（即底层存储的字符串字面量）。</font>
+ **<font style="color:rgb(0, 0, 0);">使用场景</font>**<font style="color:rgb(0, 0, 0);">：在需要原始值的操作中（如字符串拼接、比较）自动调用，优先于</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">toString()</font>`<font style="color:rgb(0, 0, 0);">。</font>
+ **<font style="color:rgb(0, 0, 0);">特点</font>**<font style="color:rgb(0, 0, 0);">：</font>
    - <font style="color:rgb(0, 0, 0);">对 String 对象调用时，直接返回原始字符串（如</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">new String('world').valueOf()</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">→</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">'world'</font>`<font style="color:rgb(0, 0, 0);">）；</font>
    - <font style="color:rgb(0, 0, 0);">与</font><font style="color:rgb(0, 0, 0);"> </font>`<font style="color:rgb(0, 0, 0);">toString()</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">结果一致，但触发时机不同：</font>`<font style="color:rgb(0, 0, 0);">valueOf()</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">侧重「获取原始值」，</font>`<font style="color:rgb(0, 0, 0);">toString()</font>`<font style="color:rgb(0, 0, 0);"> </font><font style="color:rgb(0, 0, 0);">侧重「转为字符串格式」；</font>
    - <font style="color:rgb(0, 0, 0);">原始字符串调用时，同样隐式包装后返回自身。</font>

```javascript
const strObj = new String('Hello');
console.log(strObj.toString()); // 'Hello' (返回字符串原始值)
console.log(strObj.valueOf()); // 'Hello' (返回字符串原始值)

// 自动转换
console.log(strObj + ' World'); // 'Hello World' (自动调用valueOf())
```

```javascript
function reverseString(str) {
```

```javascript
function isPalindrome(str) {
    const cleaned = str.toLowerCase().replace(/[^a-z0-9]/g, '');
    return cleaned === cleaned.split('').reverse().join('');
}

console.log(isPalindrome('A man, a plan, a canal: Panama')); // true
```

```javascript
function countCharacters(str) {

            count[char] = (count[char] || 0) + 1;
        }
    }
    return count;
}
```

### 4. URL参数解析
```javascript
function parseQueryString(query) {
    return query
        .slice(1) // 去掉开头的'?'
        .split('&')
        .reduce((params, pair) => {
            const [key, value] = pair.split('=');
            params[decodeURIComponent(key)] = decodeURIComponent(value || '');
            return params;
        }, {});
}

const query = '?name=John%20Doe&age=30&city=New%20York';
console.log(parseQueryString(query));
```

```javascript
function toCamelCase(str) {
    return str
        .replace(/[-_\s]+(.)?/g, (_, c) => c ? c.toUpperCase() : '')
        .replace(/^(.)/, (c) => c.toLowerCase());
}
```

## 性能注意事项
1. **字符串连接性能**：在循环中连接字符串时，使用数组的`join()`方法比`+=`操作符更高效

```javascript
// 低效
let result = '';
for (let i = 0; i < 1000; i++) {
    result += 'text';
}

// 高效
const arr = [];
for (let i = 0; i < 1000; i++) {
    arr.push('text');
}
const result = arr.join('');
```

2. **正则表达式预编译**：多次使用的正则表达式应该预先编译

```javascript
const regex = /pattern/g; // 预编译
str.match(regex);
```

3. **选择合适的方法**：
    - 简单查找使用`includes()`、`startsWith()`、`endsWith()`
    - 需要位置信息使用`indexOf()`、`lastIndexOf()`
    - 复杂模式匹配使用`match()`、`search()`

## 浏览器兼容性
大多数现代方法在主流浏览器中都得到良好支持：

+ ES2020方法：`replaceAll()`等较新方法需要检查兼容性

```javascript
// replaceAll的polyfill
```
